# Quickstart

Instructions for getting started with ChatAgent.

1. Install the backend:

```bash
cd backend
python -m venv .venv && source .venv/bin/activate
pip install -e .
```

2. Provide credentials and run:

```bash
echo 'CHATAGENT_GOOGLE_API_KEY=YOUR_KEY' > .env
chatagent serve
```
